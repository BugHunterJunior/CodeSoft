## Overview
A modern, dark-themed GUI calculator application built during the CodeSoft Internship, leveraging Python and Tkinter. This project provides a clean, user-friendly interface for standard arithmetic operations with interactive visual elements.

---

## Features
- **Modern UI:** Custom dark theme with contrasting accents and a flat design.
- **Interactive:** Dynamic hover color effects on keys for better user feedback.
- **Core Math:** Supports standard addition, subtraction, multiplication, and division.
- **Standalone Executable:** Available as a ready-to-run `.exe` file without needing Python installed.

---

## Project Structure
- `📁 Calculator/`: Main project folder
- `📄 main.py`: The core Python script containing the Tkinter GUI and calculation logic.
- `📄 README.md`: This documentation file.

## Installation

**Clone the repository and navigate to the project folder:**
```bash
git clone https://github.com/BugHunterJunior/CodeSoft.git
cd CodeSoft/Calculator
```

## ▶️ Usage
**Option 1: Run the Python script**
```bash
python main.py
```
