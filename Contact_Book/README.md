## Overview
A modern, fully functional GUI Contact Book application built during the CodeSoft Internship using Python, CustomTkinter, and SQLite. This project replaces basic listboxes with a clean, 2-column dashboard layout, offering secure local database storage and real-time search capabilities.

---

## Features
- **Modern UI:** 2-column layout with dark/light mode toggle, contact cards, and auto-generated avatar initials.
- **Core Database Operations:** Add, view, update, and delete contacts with duplicate phone number prevention.
- **Live Search:** Dynamic search bar that filters contacts instantly by name, phone, or email.
- **Smart Storage:** Automatically creates and utilizes a hidden system folder (`C:\ProgramData\ContactBook`) for the SQLite database.
- **Interactive Modals:** Clean popup forms for adding and editing contacts instead of cramped inline fields.

## Installation

**1. Clone the repository and navigate to the project folder:**
```bash
git clone https://github.com/BugHunterJunior/CodeSoft.git
cd CodeSoft/Contact_Book
```

**2. Install required dependencies:**
```bash
pip install customtkinter
```
