## Overview
A comprehensive task management web application built during the CodeSoft Internship, leveraging Python and Flask. This project allows users to efficiently manage their daily tasks, featuring full CRUD capabilities, secure user authentication, and persistent database storage.

---

## Features
- **:page_facing_up: Core Tasks:** Create, view, update, and delete tasks.
- **:key: Secure Auth:** User registration and login (implemented in `auth.py`).
- **:card_file_box: Database Persistence:** SQLite database with SQLAlchemy integration (managed in `database.py` and `models.py`).
- **:iphone: Responsive UI:** An intuitive and user-friendly interface across all devices.

---

## Project Structure
- `📁 static/`: (CSS, JS, and image assets)
- `📁 templates/`: (HTML templates)
- `📄 app.py`: (Application entry point)
- `📄 auth.py`: (Authentication logic)
- `📄 database.py`: (Database configuration)
- `📄 models.py`: (Database schema definitions)
- `📄 requirements.txt`: (Project dependencies)

## Installation

1. **Clone the repository and navigate to the project folder:**
   ```bash
   git clone https://github.com/BugHunterJunior/CodeSoft.git
   cd CodeSoft/todo_app
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
